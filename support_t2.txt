Tier 2: 52

Tier 2 platforms can be thought of as “guaranteed to build”. Automated tests
are not run so it’s not guaranteed to produce a working build, but platforms
often work to quite a good degree and patches are always welcome! Specifically,
these platforms are required to have each of the following:

ARM64 iOS
ARM64 CloudABI
ARM64 Android
ARM64 Fuchsia
ARM64 Linux
ARM64 Linux with MUSL
ARMv7 Android
ARMv6 Linux
ARMv6 Linux, hardfloat
ARMv6 Linux with MUSL
ARMv6 Linux, MUSL, hardfloat
ARMv5TE Linux
ARMv7 iOS, Cortex-a8
ARMv7a Android
ARMv7 CloudABI, hardfloat
ARMv7 Linux
ARMv7 Linux with MUSL
ARMv7 iOS, Cortex-a9
asm.js via Emscripten
32-bit x86 iOS
